# Use cases

## Careers submission

The careers flow is a three-step submission form for senior operators, builders, and researchers.

### Inputs

- primary company interest
- role track
- availability window
- location
- adjacent company interests
- identity and profile links
- recent role
- resume link
- highest leverage contribution
- proof of work
- optional note

### System behavior

1. Validate required fields.
2. Persist the submission.
3. Notify FrostAura support.
4. Send a branded confirmation email to the applicant.
5. Return a receipt with the submission reference and delivery status.

## Investor submission

The investor flow is a three-step expression-of-interest form for aligned capital and strategic partners.

### Inputs

- investor identity and organization
- investor type
- region
- company interests
- interest profile
- capital range
- time horizon
- preferred mode
- thesis
- value add
- optional note

### System behavior

1. Validate required fields, including at least one company interest.
2. Persist the submission.
3. Notify FrostAura support.
4. Send a branded confirmation email to the submitter.
5. Return a receipt with the submission reference and delivery status.

## Out of scope for this baseline

- Admin portal
- Authentication
- CMS authoring workflows
- Securities workflow beyond an expression-of-interest form
